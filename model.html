This webpage provides a tool for identifying sentiments across texts as "Likely Depressed" or otherwise. This tool was developed as a part of my research project on sentiment analysis. It is based on multiple Machine Learning models trained on separate datasets with over 1.6 million tweets and loads of feature engineering.
The instructions are simple, just enter the tweet in the input box below and hit predict. Don't bother removing emojis or URLs, or cleaning the text at all. The model handles that pretty well!

The model is hosted on this website. It is free to use for unlimited times, and I hope it helps you. Make sure to leave your feedback!
